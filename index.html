What is inside?

6 Modules, 39 Chapters, 70 Features and Shortcuts Covered

We believe in the concept of learning by doing. Sublime Tutor takes you methodically from the most basic commands to very advanced ones. All of this happens inside the editor itself.

Learning By Doing

Number of interesting exercises keep you entertained throughout the course. The shortcuts taught can be instantly tried out. Spaced repetition ensures that you remember them.

No Context Switching

Unlike a web based course, you don't need to switch tabs which breaks your focus. Feel like home in Sublime Text's interface and learn new commands.
